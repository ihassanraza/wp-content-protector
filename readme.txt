== Description ==

WP Content Protector is a plugin that protect your blog contents, drage image, right click and Disable keyboard shortcuts CTRL F12, CTRL+SHIFT+I CTRL, A, C, X, U, P. 

Features

-> This plugin Tested in IE9, IE10, Firefox, Google Chorme, Opera.
-> Disables right click function
-> Disables image drag and drop function
-> Disables 'Hold to Copy' function 
-> Disables keyboard copy commands (F12, CTRL+SHIFT+I CTRL, A, C, X, U, P) copy/paste/view source

== Installation ==

1. Download the .zip file
2. Upload or extract the contents of WP wp-content-protector.zip to your wp-content/plugins/folder
3. Activate the wp-content-protector plugin in your WP-admin / Plugins
